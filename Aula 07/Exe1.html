Aula 07 - Estudo de CSS

A silga CSS é Cascading ultilizada para estilizar Página da web. Ultilizando esta linguagem podemos definir:

  • Cor de fundo, texto e elementos.
  • Fonte e tamanho das letras.
  • Posicionamento de elementos dentro da pagina
  • Efeitos Visuais e animações

O CSS funciona em conjunto com o HTML, enquanto o HTML diz o que aparece, o CSS diz como deve aparecer.
